🔑 Вход через GitHub

Редактор публикует статьи напрямую в ваш репозиторий через GitHub. Это бесплатно и не зависит от Netlify. Нужен персональный токен доступа (Personal Access Token) — он сохранится только в этом браузере.

Токен хранится локально в вашем браузере (localStorage) и отправляется только на api.github.com. Мы его никуда не передаём.
📖 Как создать токен (1 раз, 2 минуты)
  1. Откройте github.com → справа вверху аватар → Settings
  2. Внизу слева → Developer settings
  3. Personal access tokens → Fine-grained tokens → Generate new token
  4. Token name: TonForce Editor
  5. Expiration: 1 год (или дольше)
  6. Repository access: Only select repositories → выберите spsonlys/tonforce
  7. Permissions → Repository permissions → Contents → Read and write
  8. Generate token → скопируйте (он показывается один раз!) и вставьте сюда

Токен с правами только на этот репозиторий. Если потеряете — просто создадите новый.
